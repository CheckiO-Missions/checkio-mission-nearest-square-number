The idea is to find square root from input number, round and square up it.
What instrument should I start from?
You should use python build-in round() function. Without second argument it rounds number to the nearest integer. 

round(2.4)
>>> 2
Ok, but what number should I round?
You need to get square root from your input number. It means that power of the number is 1/2 or 0.5. Use power operator ** also: 

round(num**0.5)
What I have to do next?
Since the function should return nearest integer to the input number (which is square number), you should get square number of the number you calculated on previous step. 

round(num**0.5)**2
How my final solution should look like?
Just add function definition: 

def nearest_square(number: int) -> int:

    return round(number**0.5)**2

def nearest_square(number):

    counter = 1
    while True:
        for sign in (1, -1):
            value = number + counter * sign
            if ((value) ** (1 / 2)).is_integer():
                return value
        counter += 1
In this solution we move from our value up and down first by 1, then 2, etc. When the square root from calculated number is integer - its out answer.
