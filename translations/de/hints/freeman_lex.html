Haben Sie eine Idee 💡! Um die Quadratwurzel aus der eingegebenen Zahl zu ermitteln, runden Sie sie und bringen sie ins Quadrat.

Von welchem Instrument soll ich ausgehen?

Sie sollten die in Python eingebaute Funktion round() verwenden. Ohne zweites Argument rundet sie die Zahl auf die nächste Ganzzahl. 

round(2.4) == 2
Ok, aber welche Zahl soll ich runden?

Sie müssen die Quadratwurzel aus Ihrer Eingabezahl ziehen. Das bedeutet, dass die Potenz der Zahl ist 1/2 oder 0,5. Verwenden Sie auch den Potenzoperator **: 

round(num**0.5)
Was muss ich als nächstes tun?

Da die Funktion die nächste Ganzzahl zur eingegebenen Zahl zurückgeben soll (die eine Quadratzahl ist), sollten Sie die Quadratzahl der Zahl erhalten, die Sie im vorherigen Schritt berechnet haben. 

round(num**0.5)**2
Wie sollte meine endgültige Lösung aussehen?

Fügen Sie einfach eine Funktionsdefinition hinzu: 

def nearest_square(number: int) -> int:

    return round(number**0.5)**2

Doppelte Verwendung der Funktion pow() aus dem vorimportierten Modul math. Pow steht für 'Potenz' und benötigt zwei Argumente: Zahl und Potenz, auf die die Zahl hochgerechnet werden soll. 

from math import pow

nearest_square = lambda num: pow(round(pow(num, 0.5)), 2)

In dieser Lösung bewegen wir uns von unserem Wert nach oben und unten, zuerst um 1, dann um 2 usw. Wenn die Quadratwurzel aus der berechneten Zahl eine ganze Zahl ist, erhalten Sie die Antwort. 

def nearest_square(number):

    counter = 1
    while True:
        for sign in (1, -1):
            value = number + counter * sign
            if ((value) ** (1 / 2)).is_integer():
                return value
        counter += 1
