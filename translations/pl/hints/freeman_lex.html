Proszę złapać pomysł 💡! Aby znaleźć pierwiastek kwadratowy z liczby wejściowej, proszę ją zaokrąglić i podnieść do kwadratu.

Od jakiego instrumentu powinienem zacząć?

Powinien Pan użyć wbudowanej funkcji Pythona round(). Bez drugiego argumentu zaokrągla ona liczbę do najbliższej liczby całkowitej. 

round(2.4) == 2
Ok, ale jaką liczbę powinienem zaokrąglić?

Musi Pan otrzymać pierwiastek kwadratowy z liczby wejściowej. Oznacza to, że potęga liczby wynosi 1/2 lub 0,5. Proszę również użyć operatora potęgi **: 

round(num**0.5)
Co mam dalej zrobić?

Ponieważ funkcja powinna zwrócić liczbę całkowitą najbliższą liczbie wejściowej (która jest liczbą kwadratową), powinien Pan otrzymać liczbę kwadratową liczby, którą Pan obliczonej w poprzednim kroku. 

round(num**0.5)**2
Jak powinno wyglądać moje ostateczne rozwiązanie?

Proszę dodać definicję funkcji: 

def nearest_square(number: int) -> int:

    return round(number**0.5)**2

Podwójne użycie funkcji pow() z wstępnie zaimportowanego modułu matematycznego. Pow oznacza "potęgę" i przyjmuje dwa argumenty: liczbę i potęgę, do której ma zostać podniesiona liczba. 

from math import pow

nearest_square = lambda num: pow(round(pow(num, 0.5)), 2)

W tym rozwiązaniu przechodzimy od naszej wartości w górę i w dół najpierw o 1, potem o 2 itd. Gdy pierwiastek kwadratowy z obliczonej liczby jest liczbą całkowitą - jest to nasza odpowiedź. 

def nearest_square(number):

    counter = 1
    while True:
        for sign in (1, -1):
            value = number + counter * sign
            if ((value) ** (1 / 2)).is_integer():
                return value
        counter += 1
