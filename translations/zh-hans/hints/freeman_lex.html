双重使用预导入数学模块中的 pow() 函数。Pow 代表 "幂"，有两个参数：数值和数值的幂。 

from math import pow

nearest_square = lambda num: pow(round(pow(num, 0.5)), 2)

在本解 决方案中，我们首先将数值向上和向下移动 1，然后是 2，等等。当计算出的数值的平方根为整数时，答案就出来了。 

def nearest_square(number):

    counter = 1
    while True:
        for sign in (1, -1):
            value = number + counter * sign
            if ((value) ** (1 / 2)).is_integer():
                return value
        counter += 1
