Ідея у тому, щоб знайти квадратний корінь вхідного числа, округлити його і звести до квадрату.
З якого інструменту мені слід розпочати?
Тобі варто використати вбудовану функцію round(). Без другого параметру вона округлює до найближчого цілого. Саме те, що тобі потрібно! 

round(2.4)
>>> 2
Добре, але яке число я маю заокруглювати з її допомогою?
Тобі потрібно знайти квадратний корінь вхідного числа. Це значить, що число підноситься до ступеня 1/2 або 0.5. Також використай оператор зведення до степеня **: 

round(num**0.5)
Що мені робити далі?
Оскільки функція повинна повертати найближче ціле число до вхідного числа (яке є квадратним числом), ти маєш отримати квадрат числа, яке ти розрахував на попередньому кроці. 

round(num**0.5)**2
То як має виглядати моє остаточне рішення?
Просто додай визначення функції: 

def nearest_square(number: int) -> int:

    return round(number**0.5)**2

from math import pow

    nearest_square = lambda num: pow(round(pow(num, 0.5)), 2)
Подвійне використання функції pow() з попередньо імпортованого модуля math. Pow скорочено від 'power' і приймає два аргументи: число і степінь, до якого його піднести. 

def nearest_square(number):

    counter = 1
    while True:
        for sign in (1, -1):
            value = number + counter * sign
            if ((value) ** (1 / 2)).is_integer():
                return value
        counter += 1
У цьому розв'язку ми рухаємося від нашого значення вгору і вниз спочатку на 1, потім на 2 і т.д. Коли корінь квадратний з розрахованого значення є цілим числом - це і буде наша відповідь.
